  ## 项目简介
    一个商业航天载荷地面主控程序示例，目前属于某商业航天公司在役软件，主要功能包括指令配置发送、遥测接收解析、图像显示处理等。支持串口、CAN、LVDS通信
    方式。
      
  ## 开发环境
    Based on Qt 5.12.3(MSVC 2017, 32bit)
    Qt Creator 4.9.1
    
  ## 依赖安装
   1. 下载 Lua 5.4.2 库：[官方链接](https://www.lua.org/download.html)
   2. 将库文件放入 `satellitePayloadGroundControl/frameFormat/lua/` 目录。
